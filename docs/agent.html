Anthropic 2024年11月发布的开放协议。定义AI Agent与外部工具/数据的标准通信方式。2025年底 Anthropic 发布 Agent Skills 开放标准 (.skill.md)，OpenAI 随后为 Codex CLI 采纳同一格式。已成为事实标准。

A2A — Agent-to-Agent Protocol

Google 2025年推出的Agent间通信协议。允许不同平台的Agent互相发现和协作。与MCP互补：MCP解决Agent↔工具连接，A2A解决Agent↔Agent通信。两者正在通过 Agentic AI Foundation 走向融合。
